### 用户登录

**功能描述**：在用户进入程序时调用，记录用户状态

**请求链接**：`/user/login`

**请求方式**：POST

**参数详情**

| 参数 | 类型   | 是否必需 | 说明     |
| ---- | ------ | -------- | -------- |
| id   | string | 是       | 用户标识 |
| name | string | 是       | 用户名   |

示例

``` json
{
    "id":"11111",
    "name":"CHEN YANG"
}
```



**返回参数**

| 参数 | 类型   | 说明                                            |
| ---- | ------ | ----------------------------------------------- |
| code | number | 状态码，200表示正常，400参数异常，500服务器异常 |
| msg  | string | 提示信息                                        |

示例

```json
{
  "code": 200,
  "msg": "success"
}
```





### 用户登出

**功能描述**：在用户退出程序时调用，释放资源

**请求链接**：`/user/logout`

**请求方式**：GET

**返回参数**

| 参数 | 类型   | 说明                                            |
| ---- | ------ | ----------------------------------------------- |
| code | number | 状态码，200表示正常，400参数异常，500服务器异常 |
| msg  | string | 提示信息                                        |

示例

```json
{
  "code": 200,
  "msg": "quit"
}
```





### 历史记录

**功能描述**：获得用户十天内的历史记录图片

**请求连接**：`/user/history`

**请求方式**：GET

**发送参数**：无

**返回参数**

| 参数 | 类型   | 说明                                             |
| ---- | ------ | ------------------------------------------------ |
| code | number | 200 正常，400 用户不合法，500 数据库异常         |
| msg  | string | 补充信息                                         |
| data | array  | json数组，内含timestamp保存的时间戳，url图片地址 |

示例

```json
{
    "code": 200,
    "data": [
        {
            "timestamp": "1612779210",
            "url": "1257.0.0.1:3268/imge.jpg"
        },
        {
            "timestamp": "1612777913",
            "url": "127.0.0.1:3268/se.jpg"
        }
    ],
    "msg": "success"
}
```

### 青山绿水图

**功能描述**：获得青山绿水风格的迁移后的图片

**请求链接**：`/transfer/style_qlssh_no`

**请求方式**：POST

**参数详情**

| 参数  | 类型   | 是否必需 | 说明           |
| ----- | ------ | -------- | -------------- |
| id    | string | 否       | 用户标识       |
| alpha | string | 是       | 模型参数       |
| image | string | 是       | 图片base64之后 |

示例

``` json
{
    "id":"11111",
    "alpha":"0.5",
    "image":"data:image/jpg;base64,/9j/4AAQSkZJRgABAQAAAQABAAD/2w..."
}
```

**！注意：如果不传参数id，图片不会入库**



**返回参数**

| 参数 | 类型   | 说明                                            |
| ---- | ------ | ----------------------------------------------- |
| code | number | 状态码，200表示正常，400参数异常，500服务器异常 |
| msg  | string | 提示信息                                        |
| url  | string | 迁移后图片的地址                                |

示例

```json
{
  "code": 200,
  "msg": "success without id",
  "url": "image/url.jpg"
}
```



**！其他三个和这个类似。**





# 数据库

**表**

user(id pk)：记录用户

image(id  pk, url)：记录图片

transfer(user_id, image_id, timestamp)：记录用户转换的图片
